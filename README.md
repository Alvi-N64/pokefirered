# Pokémon FireRed and LeafGreen

This is an Indonesian translation of Pokémon FireRed and LeafGreen based on the pret decompilation.

Kalau judulnya diterjemahin mungkin jadi 
Pokémon Merapi (Merah + Api) dan Pokémon Hidaun (Hijau + Daun). 😅


# Rencana

- Pokédex nya pakai Metric (seperti di TCG Indonesia & kehidupan sehari-hari)
- Ada 2 patch: yang satu melokalisasi nama-nama Pokémon & kota-kota nya ke Indonesia, yang satunya ngikut nama-nama versi English
